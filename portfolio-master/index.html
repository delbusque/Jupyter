About Me

Hi, I am Ventsislav and enthusiastically deep dived into Web Development. I have started programming in 2020 at Software University and graduated Programming Basics and Fundamentals, as well as Advanced and OOP modules with C#. Afterwards, I turned around equipped with the fundamentals into functional programming with JavaScript following the university curriculum of JS Advanced, JS Apps and JS Back-End.

In that way I gained knowledge for DOM Manipulations and Events, HTTP and REST Services, Asynchronous Programming, CRUD, SPA with client-side as well as MPA with server-side rendering. I met templating engines like Handlebars and lit-html, routing concepts, Node.js with ExpressJS, MongoDB with the Mongoose models and querying, Authentication concepts, Error Handling, Validation and Unit testing.

Nowadays I am graduating from the latest editions of React JS and HTML & CSS modules with their hooks, routing, grid, flex and responsive design concepts. Also interact with the jQuery library, Bootstrap and SASS extension language.

I am grateful to my teachers and mentors for my attitude till this moment. But I look towards the future and I am sure that participating in professional projects and real-life scenarios is the perfect opportunity to continue that never ending Web Development journey.
